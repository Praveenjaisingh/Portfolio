## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/praveenjaisingh/portfolio.git
```

### Navigate to the Project

```bash
cd portfolio
```

### Install Dependencies

```bash
npm install
```

### Run the Development Server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
